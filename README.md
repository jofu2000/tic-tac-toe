Tic-Tac-Toe (C)



Console implementation of the classic Tic-Tac-Toe game written in C.



Description:

This project is a simple terminal-based Tic-Tac-Toe game for two players.

The game is played on a 3x3 board. Players take turns placing their symbols until one of them wins or the game ends in a draw.



Features:



Two-player game on one computer



Turn-based gameplay



Input validation



Win condition checking



Draw detection



Text-based game board



Build and Run:

Build: gcc src/main.c -o tic\_tac\_toe

Run: ./tic\_tac\_toe (On Windows: tic\_tac\_toe.exe)



Gameplay:



Player X always starts



Players enter a cell number from 1 to 9



The game ends when a player wins or the board is full
